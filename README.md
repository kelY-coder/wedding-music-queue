# 💚 Wedding Music Queue System

Complete real-time music queueing system for weddings with guest requests, DJ controls, NFC integration, and rustic branding.

## Features

✅ **Shareable Guest Portal** - Guests add songs from any device  
✅ **Real-time Queue** - WebSocket updates for instant sync  
✅ **NFC Integration** - Tap-to-add music with NFC tags  
✅ **DJ Control Panel** - Manage queue, skip songs, remove duplicates  
✅ **Now Playing Display** - Shows current song & requester  
✅ **Custom Branding** - Rustic wedding-themed UI with monogram  
✅ **Mobile Responsive** - Works perfectly on phones  
✅ **1-Click Deployment** - Railway.app ready
